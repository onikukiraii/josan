# 産婦人科シフト管理アプリ 概要

作成日: 2026-02-22
ステータス: ドラフト

---

## 概要

産婦人科クリニックにおけるスタッフのシフト管理を自動化するWebアプリケーション。約30名規模のスタッフに対し、月単位でシフトを自動生成し、各種制約条件を満たしながら公平な勤務割り当てを実現する。

## 主要機能

- シフト自動生成（制約充足ソルバーによる最適化）
- 希望休の提出・管理
- メンバー登録・属性管理
- シフト表の表示・編集

## 技術スタック

- バックエンド: FastAPI（Python）
- フロントエンド: React + TypeScript + Vite
- シフト生成エンジン: Google OR-Tools CP-SAT ソルバー
